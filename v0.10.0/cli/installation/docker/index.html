    Docker

    We also release Docker images aquasec/starboard:0.10.0 and public.ecr.aws/aquasecurity/starboard:0.10.0 to run Starboard as a Docker container or to manually schedule Kubernetes scan Jobs in your cluster.

    $ docker container run --rm public.ecr.aws/aquasecurity/starboard:0.10.0 version
